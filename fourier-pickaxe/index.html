Technical idea

The function payload stores only G0 palette indexes and box-basis bytes. Later miners can use the exact model hash as the target and search for a shorter Fourier gene.

  1. Load VOX and canonicalize its voxel/color grid.
  2. Map colors to the shared G0 palette and merge same-color voxels into box basis functions.
  3. Evaluate the function bytes and redraw the model from G0 palette indexes.
  4. Use the exact result hash as a future PoW target for shorter genes.
