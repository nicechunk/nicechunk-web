NCK Economy

NCK powers the NiceChunk economy.

NCK is designed as the core utility token of NiceChunk. It can be used for genesis access, in-world actions, forging systems, marketplace settlement, and future governance mechanics.

Planned Utility

  • Genesis Pass access connected to NCK
  • In-world action and material loops
  • Forging systems and asset creation
  • Marketplace settlement
  • Future world governance mechanics

Watcher Network

Run a Watcher. Keep a region alive.

Watcher nodes are the access layer of NiceChunk. They do not control player assets or process on-chain transactions.

Watcher Node Region access layer

Open World Rules

Transparent world rules.

NiceChunk is built around open generation rules. Chunks are generated from seeds, player actions update world state, and important economic actions are designed to be verifiable.

Roadmap

From alpha world preview to player-owned protocol.

  1. Phase 1 - Alpha World Preview

    Browser voxel client, wallet connection, seeded terrain preview, player setup.

  2. Phase 2 - Mining & Forging

    Resource discovery, block interaction, material logic, forging bench, basic asset loop.

  3. Phase 3 - NCK Economy

    Genesis pass, NCK utility, marketplace settlement, player-owned asset layer.

  4. Phase 4 - Watcher Network

    Region access nodes, real-time communication, watcher staking, region-level participation.

  5. Phase 5 - World Governance

    Territory rules, world parameters, player voting, long-term protocol governance.
