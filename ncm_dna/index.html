NCM DNA Lab

NCM DNA Lab is an experimental gene search system for on-chain 3D assets.

In the future, this search can become a useful Proof of Work.

Raw Cuboid Model ↓ NCM-DNA Gene Search ↓ Grow(gene) Hash Match

TARGET MODEL

BEST RESULT

Current shortest matching gene

3D PREVIEW

Canonical cuboid model

CPU FRIENDLY

Why CPU-first?

NCM-DNA v1 uses a rule-based cuboid growth language instead of GPU-friendly voxel functions.

FUTURE ON-CHAIN FLOW

Storage-saving submission path

  1. Asset PDA stores canonical_model_hash and current_encoding.
  2. Miner submits a shorter gene.
  3. Program verifies grow(gene) hash equals canonical_model_hash.
  4. If gene is shorter, asset encoding is updated.
  5. Released PDA rent can be shared between miner, owner, and protocol pool.
