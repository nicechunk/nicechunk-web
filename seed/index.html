Fair By Code

A public voxel world should not depend on hidden asset servers.

NiceChunk's core claim is simple: resources originate from the public world map, materials come from resource smelting, items come from material forging, and ownership should be inspectable on-chain.

No IPFS

No external gateway is required to define the source of game resources or item identity.

No Centralized Asset Storage

The goal is chain-readable assets and deterministic reconstruction, not private hosted metadata.

No Team-Minted Resources

The team cannot create arbitrary resources or items outside the resource, smelting, and forging path.

Assets / Gameplay / User Data On-Chain

The important state surface is designed to be public, replayable, and wallet-owned.
